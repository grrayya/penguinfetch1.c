# penguinfetch

ute system information fetch tool written in C. 

Penguinfetch is a small program that shows your computer's stats next to a text-art penguin.

What it includes:
Speed: It is written in C, so it loads instantly.
File-safe: It removes color codes automatically if you save the output to a text file.
System info: It displays your user, OS, kernel, uptime, RAM, CPU, and shell.
Settings: You can add command-line options to hide the penguin or turn off the colors.

Installation
You will need `gcc` and `make` installed on your system.

1. Clone or download this repository.
2. Compile the binary:
   ```bash
   make
